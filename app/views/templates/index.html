Spotify Color API

NowPlaying et couleurs, sans blabla.

Créer un compte Se connecter

Configurer en 3 étapes

  1. Créez un compte puis connectez‑vous.
  2. Allez sur /<UUID>/settings et faites:
    • Renseignez votre Client ID Spotify
    • Autorisez l’accès à Spotify
  3. C’est prêt : utilisez vos pages publiques et l’API.

Vos URLs (exemples)

Remplacez <UUID> par le vôtre (visible après connexion).

Pages publiques

  • /<UUID>/nowplayingOverlay NowPlaying
  • /<UUID>/color-fullscreenPlein écran couleur

API JSON

  • /<UUID>/infosInfos et état Spotify
  • /<UUID>/colorCouleurs extraites

Pages privées

  • /<UUID>Profil (propriétaire seulement)
  • /<UUID>/settingsRéglages Spotify (propriétaire)
